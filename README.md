# chat-app-awesome
Personal chat-app-awesome collection

## Overview
This project is a personal collection of chat applications developed for learning purposes.

## Features
- Real-time messaging
- User authentication
- Multiple chat rooms

## Tech Stack
- Node.js
- Express
- Socket.io

## Getting Started
1. Clone the repository
2. Install dependencies: `npm install`
3. Start the server: `npm start`
